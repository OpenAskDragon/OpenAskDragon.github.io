# Triggs 等《Bundle Adjustment — A Modern Synthesis》：可写作证据提纲

## 文献与版本核验

- **书目信息**：Bill Triggs, Philip F. McLauchlan, Richard I. Hartley, Andrew W. Fitzgibbon, “Bundle Adjustment — A Modern Synthesis,” *Vision Algorithms: Theory and Practice*, LNCS 1883, Springer, 2000, pp. 298–372, DOI `10.1007/3-540-44480-7_21`。出版版首页（`Triggs00.pdf`，PDF 第 1 页／LNCS p. 298）给出作者、卷号、页码和年份；HAL 版本扉页（`bundleadjustment.pdf`，PDF 第 1 页）给出会议、卷号、页码、DOI 与 HAL 标识。
- **版本关系**：两份文件均为四位作者的同一论文，标题与摘要相同。`Triggs00.pdf` 为 75 PDF 页的 LNCS 排版本（印刷页 298–372）；`bundleadjustment.pdf` 为 72 PDF 页的 HAL 存档本，前置存档/引文页并采用不同的 PDF 分页。后者的论文首页在 PDF 第 2 页，摘要随正文出现；该差异只应作为文档版本/分页说明，不能写成观点或结论的差异。
- **引用约定**：下文优先使用稳定的 LNCS 印刷页码，并附 `Triggs00.pdf` PDF 页；HAL 定位仅用于版本核验。

## 可用于文章的论证骨架

### 1. 开篇：它要解决的是什么问题

- **可写主张**：Bundle adjustment 是把视觉重建中的 3D 结构与相机视角参数（位姿及/或标定）一起优化，使其在所选代价函数下联合最优；“bundle”指从 3D 特征通往相机中心的光线束。
  - **依据**：摘要及 §1 Introduction，LNCS pp. 298–299（`Triggs00.pdf` PDF pp. 1–2）。
  - **限定**：将“最优”解释为“相对于明示的模型拟合代价最小”，不要表述为无条件的真实世界最优。
- **可写主张**：作者把它定位为大型、稀疏的几何参数估计问题，核心变量是 3D 特征、相机位姿和标定；迁移到其他测量问题时，关键是选择能利用结构和稀疏性的数值优化法。
  - **依据**：§1，LNCS p. 299（PDF p. 2）。

### 2. 历史动机：不是“新发明”，而是知识迁移

- **可写主张**：论文面向计算机视觉实现者，称其结果大多早已出现在摄影测量和大地测量文献中，但在视觉领域尚不广为人知，目标是减少重复发明、纠正误解、促进两个社群交流。
  - **依据**：§1，LNCS p. 298（PDF p. 1）。
- **可写主张**：历史附录将最小二乘追溯到 Gauss/Legendre（约 1795–1820），并把摄影测量 bundle 方法的早期电子计算机实现归于 Duane C. Brown 团队（1957–59）。
  - **依据**：Appendix A “Historical Overview”，LNCS p. 352（PDF p. 55）。
  - **限定**：这是作者的简要历史概述，写作时应归因于该文，不扩展为未经另行考证的完整学科史。

### 3. 论文覆盖范围：给文章的章节地图

- **可写主张**：论文的明确覆盖声明依次包括投影/参数化（§2）、误差度量（§3）、优化理论（§4）、网络结构与特征稀疏性（§5）、二阶/一阶/递推实现策略（§§6–8）、gauge/datum 自由度（§9）、质量控制（§10）和网络设计（§11）；§12 给出暂定建议。
  - **依据**：§1 “Coverage”，LNCS pp. 300–301（PDF pp. 3–4）。

### 4. 方法主线：从模型到求解

- **参数化**：在局部步长预测中避免奇异、无穷远、强非线性和病态；对旋转可采用残差旋转或四元数，对远处点/线/面可采用齐次射影参数化。
  - **依据**：§2.2，LNCS pp. 302–304（PDF pp. 5–7）；§12，p. 350（PDF p. 53）。
- **robust（鲁棒代价）**：作者主张代价应近似总误差（inlier + outlier）分布的负对数似然；传统最小二乘因高斯尾部过小而对离群值敏感。鲁棒分组应对应实际会共同失效的观测单位。
  - **依据**：§3、§3.3，LNCS pp. 304, 309–310（PDF pp. 7, 12–13）；§12，p. 350（PDF p. 53）。
  - **限定**：文中说精确的分布形式“不太关键”，但 inlier 协方差/离散度要现实；不可把它简化成“任意 M-estimator 都一样”。
- **sparse（二阶与稀疏求解）**：二阶 Newton 风格方法以快速渐近收敛和较高单步成本为特征，效率依赖利用 Hessian 的网络结构和稀疏性；批处理问题的暂定建议是 Gauss–Newton 加 Hessian 稀疏分解。
  - **依据**：§6、§6.1，LNCS p. 320（PDF p. 23）；§12，p. 351（PDF p. 54）。
  - **可解释但须归因**：Schur complement/reduced bundle system 是文中二阶实现策略的一部分（§6.1），不是本文发明的现代框架。
- **gauge（坐标/基准自由度）**：gauge freedom 是理论问题；高成本的稠密分解或伪逆不宜直接用于固定 gauge。作者建议以低成本工作 gauge 计算，再用 gauge projector 转到所需输出 gauge；除非需要外部参考，输出可选以关心的网络元素为中心的 inner gauge。
  - **依据**：§9，尤其 §9.5，LNCS pp. 335–343（PDF pp. 38–46）；§12，p. 352（PDF p. 55）。

### 5. 可靠性不是优化后的附属品

- **quality（质量控制）**：作者把质量控制定义为用诊断测试检测离群值，并刻画参数估计的总体准确性与可靠性；其简洁准则为“quality = accuracy + reliability”。
  - **依据**：§10、§10.1，LNCS pp. 343–344（PDF pp. 46–47）。
- **network（网络设计）**：网络设计是在测量前规划相机位置和图像数量，以获得足够准确、可靠的估计；量化辅助包括在合适 gauge 下的协方差估计和质量控制测试。作者特别强调子网络重叠、宽基线/会聚几何以及冗余；经验规则是每个特征点通常至少见于四幅图像。
  - **依据**：§11，LNCS pp. 349–350（PDF pp. 52–53）。
- **收束性建议**：鲁棒代价有帮助，但不足以替代事前测量规划和事后对离群敏感性、过度建模的检查；低冗余、弱几何或过于一般的模型会使严重错误难以察觉。
  - **依据**：§12，LNCS p. 352（PDF p. 55）。
